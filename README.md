# Neetinder(似ティンダー)
理想の相手を探し回るのはもうやめませんか？
NeetinderはAIを使ってタイプ有名人に似た人だけをユーザーのなから探し出してくれる出会いアプリです。有名人との顔の類似度を計算し類似度の高い３名だけを表示するので、他のアプリでスワイプに疲れたあなたにとって素敵な体験となるでしょう。
※デモのユーザーは仮想のユーザーです
